# Available Commands

These commands are available as workflows:

## /commit

**Source:** `.agent/commands/commit.md`

Create atomic git commits with conventional messages

## /debug

**Source:** `.agent/commands/debug.md`

Systematic debugging with root cause analysis and fixes

## /document

**Source:** `.agent/commands/document.md`

Create or update documentation with research, planning, and review

## /gather-context

**Source:** `.agent/commands/gather-context.md`

Gather context about repository state and recent changes

## /init

**Source:** `.agent/commands/init.md`

Initialize AGENTS.md file with best practices for agentic development

## /preset-help

**Source:** `.agent/commands/preset-help.md`

Explain what this preset provides and how to use it

## /refactor

**Source:** `.agent/commands/refactor.md`

Large-scale refactoring with phased execution and validation

## /research

**Source:** `.agent/commands/research.md`

Research codebase and external docs with parallel analysis

## /review

**Source:** `.agent/commands/review.md`

Comprehensive code review with testing and quality checks

## /test

**Source:** `.agent/commands/test.md`

Write comprehensive tests with TDD or verification approach
